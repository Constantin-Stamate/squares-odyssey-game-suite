## Overview

**The Squares' Odyssey** is an interactive game suite that reimagines the spirit of the Ancient Olympic Games through a trilogy of modern, logic-based challenges. Rooted in Greek mythology and inspired by the competitive essence of Olympia, this application transforms traditional athletic values into mental feats of strength, agility, and precision.
<p align="center">
  <img src="src/main/resources/static/images/description/temple.png" alt="Divinity page"/>
</p>

## Features

- **Modular Design**: Structured for easy maintenance and future updates.

- **Game Selection**: Choose between three gods, each representing a unique challenge.
<p align="center">
  <img src="src/main/resources/static/images/description/selection.png" alt="Selection page"/>
</p>

- **Battle of Gods**: Guess a secret 4-digit code within 7 attempts.
<p align="center">
  <img src="src/main/resources/static/images/description/divinity.png" alt="Divinity page"/>
</p>

- **Sliding Puzzle**: Rearrange tiles to complete an image; unsolvable puzzles are detected.
<p align="center">
  <img src="src/main/resources/static/images/description/puzzle.png" alt="Puzzle page"/>
</p>

- **Memory Mastery**: Match all card pairs within 15 attempts.
<p align="center">
  <img src="src/main/resources/static/images/description/card.png" alt="Memory page"/>
</p>

- **Reward System – Success**: Earn medals and see festive animations when you win a challenge.
<p align="center">
  <img src="src/main/resources/static/images/description/medal-battle.png" alt="Medal Battle page" style="width: 700px; height: auto;"/>
</p>

- **Reward System – Failure**: Themed failure screens with encouraging messages motivate you to try again.
<p align="center">

## Installation

To install the application, follow these steps:

1. **Clone this repository:**

```bash
  git clone https://github.com/Constantin-Stamate/TheSquaresOdyssey
```

2. **Navigate into the project directory:**

```bash
  cd TheSquaresOdyssey
```

3. **Run the application:**

```bash
  mvn javafx:run
```
